ambari-presto-service¶

Issues | Github

Introduction¶

This project contains the code and configuration needed to integrate Presto with Ambari. Adding the Presto service to Ambari allows:

  1. Installing and deploying Presto on a cluster from the Ambari UI.
  2. Changing Presto configuration options via the Ambari UI.
